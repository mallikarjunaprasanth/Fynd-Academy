step -1   create folder in vs code 

    1. git init                                     -- initialzing repo 

    2. git remote add origin             -- Connecting repo

    3. git branch - checking branch                 -- Checking branch

    4. git checkout -b             -- Create a new branch 


 step-2    modify code 
    
    5. git add .                                    -- All changed files are added

    6. git status                                   -- Checking the changed file

    8. git commit -m "commit code"                  -- Commit code

    9. git pull origin branch/master                -- Pull code 


step-3    follow step-2 again

    10. git push origin branch/master 

             (or)

    11. git merge master branch                     -- Merging master to branch

follow point 10 again to push code after merge


    12. git pull                                    -- fetch code from branch/master

you can see total code with changes here


Step -4  Delete remote branch

    13. Delete branch remote                        --git push origin --delete newBranch
