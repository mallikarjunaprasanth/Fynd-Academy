Anchor tag

The a-tag defines a hyperlink, which is used to link from one page to another page

    Attributes of Anchor tag :

  1.  _blank
  2.  _parent
  3.  _self
  4.  _top
  5.  _frame name
_blank

target _blank link opens always new tab


_parent

target _parent link opens in same tab


_self

target _self link opens in same tab


_top

target _top link opens in same tab


framename

target framename link opens always new tab
